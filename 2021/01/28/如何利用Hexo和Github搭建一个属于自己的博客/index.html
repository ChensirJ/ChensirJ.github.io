如何利用Hexo和Github搭建一个属于自己的博客（1）

准备工作

准备工作：

你需要用到东西有：

  • Github账号
  • Node.js
  • Git

Github账号

如果没有Github账号的话，可以先去Github官网注册一个账号，注册的过程也非常的简单，填写一些基本信息就可以完成注册了

注册完成了之后，点击Your Repositories，进入你的仓库页面

点击new，新建一个仓库，点击之后可能提示需要验证邮箱，这时候打开你注册时候填的邮箱，找到验证邮件，点击验证就好了。

仓库的名字应该设置为 你的github用户名.github.io，就像下面这样

然后点击创建按钮就好了

至此，Github上的工作告一段落了

Node.js

进入Node.js官网，下载最新版的Node.js

下载完成之后进行安装

安装完成之后打开命令提示符(cmd)，输入node -v，如果出现版本号，则说明安装成功

至此Node.js的工作也结束了

进入Git官网，下载最新版的Git
